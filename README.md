# Sistema de Ordem de Serviço (OS)

## Descrição
Sistema desenvolvido em Python para gerenciamento de ordens de serviço, clientes, técnicos e serviços. Permite cadastrar, consultar e apagar registros de forma interativa via terminal.

---

## Funcionalidades
- Cadastro de clientes e técnicos
- Criação de novas ordens de serviço
- Cadastro de serviços e itens de OS
- Consultas detalhadas por ID
- Exclusão de clientes e serviços
- Menu interativo com validação de entradas
- Feedback colorido no terminal
